Learning Continuous Image Representation with Local Implicit Image Function Chen et al., CVPR 2021

This paper presents LIIF, which represents an image as a continuous function. It's particularly useful for super-resolution tasks because it allows for querying at arbitrary coordinates. The integration of a Fourier feature network significantly improves the rendering of high-frequency details, which is highly applicable for restoring fine structures.
